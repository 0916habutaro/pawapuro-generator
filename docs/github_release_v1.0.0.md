# パワプロ選手ジェネレーター v1.0.0

## 主な機能

- ローカル用Streamlitアプリとして、架空のパワプロ風選手を生成できます。
- 投手・野手を選び、能力カード形式で結果を確認できます。
- 名前、国籍、地域、年齢、投打、ポジション、基礎能力、能力ランク、特殊能力、ランク系、変化球、投手適正、サブポジションを自動生成します。
- 実在選手の完全再現を目的としたものではありません。

## 生成カテゴリ

- 架空球団用
- ドラフト候補用
- 助っ人外国人用

## 保存・履歴

- 生成結果はリポジトリ直下のSQLite DB `players.sqlite3` に保存されます。
- 履歴画面で過去生成選手を確認できます。
- 履歴はCSV・Excel形式で出力できます。

## バランス確認

- 保存済み選手の人数、カテゴリ、能力平均、特殊能力、ランク系、変化球などを確認できます。
- 少数サンプルでは分布が偏ることがあります。

## セットアップ

Python仮想環境の利用を推奨します。

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
pip install -r requirements.txt
streamlit run app.py
```

## 既知の仕様

- usageは生成対象外です。
- 金特は生成対象外です。
- 生成結果はランダムであり、同名選手が低確率で重複する可能性があります。
- このアプリはローカル利用を前提としており、外部APIは不要です。

## 更新時の注意

- 更新前にStreamlitを停止してください。
- 更新前にSQLite DB `players.sqlite3` をバックアップしてください。
- DBは生成履歴の本体です。削除すると同じ履歴は復元できません。
