# 投手基本能力 役割別調整レビュー（after）

## 結論

架空球団用投手は、球速・コントロール・スタミナの役割別平均差分が合格目安内に収まったため、**投手基本能力は完成扱いにしてよい**と判断します。変化球生成、投手適正パターン、野手生成ロジックは変更していません。

## 修正したロジック

- `generate_pitcher_abilities` の基本能力生成後、最終 `ability()` 化の直前に、カテゴリ・主起用・選手タイプ・適正段階に応じた軽い補正を追加しました。
- 架空球団用は先発/中継ぎ/抑えの差分縮小を主目的に、球速・コントロール・スタミナのみ補正しました。
- ドラフト候補用は平均を実在並みにせず、速球派・年長候補・抑え候補のレア枠だけ残る程度の補正に留めました。
- 助っ人外国人用は中継ぎの万能化を抑えるため、主にコントロールとスタミナを抑えました。
- 補正は新しい乱数を消費しない決定的な計算にしており、球種抽選・方向抽選・第二球種抽選の乱数列を動かさないようにしています。

## 分位点の変化

- 先発は球速平均が before 148.793 → after 149.965、コントロール 51.832 → 54.775、スタミナ 61.463 → 57.704 となり、中央値・上位分位点も実在側へ寄りました。
- 中継ぎは球速 149.990 → 151.807、コントロール 52.337 → 47.622、スタミナ 50.905 → 47.895 となり、制球型を残しながら万能化を抑制しました。
- 抑えは球速 151.432 → 153.974、コントロール 54.423 → 52.075、スタミナ 45.860 据え置きで、速球派上位層を増やしつつ変化球型・高制球守護神の余地を残しました。

## レア投手タイプの維持

- 速球派補正を役割別に強めたため、抑え・中継ぎの高球速レンジは維持されています。afterの架空球団用投手全体の最大球速は165km/h、抑え平均差分は -0.776km/h まで縮小しました。
- 技巧派はコントロール低下を弱め、制球型中継ぎ・高制球守護神が消えないようにしました。
- スタミナ型と先発○/ロングリリーフ相当はスタミナ低下を弱め、先発上位・ロングリリーフ型を残しています。

## ドラフト候補用への影響

- ドラフト候補用は実在平均に合わせに行かず、抑え候補や年長速球派に限定して球速を補正しました。
- 中継ぎ・先発のスタミナ過多は軽く抑えていますが、即戦力先発、上位指名級の高球速投手、将来の守護神候補、高校生・大学生の突出素材型が残る余地を維持しています。

## 助っ人外国人用への影響

- 助っ人外国人用は球速・変化球の尖りを維持しつつ、中継ぎのコントロール/スタミナを下げて万能化を抑えました。
- 先発には不足気味だったコントロールを少し補い、抑えはスタミナを動かさず高球速寄りの特徴を維持しています。

## 変化球分布が変わっていないこと

- 球種数、総変化量、第二球種率は before / after で一致しています。
- 新しい乱数消費を追加していないため、球種方向分布・球種名分布・第二球種判定・投手適正パターン分布は意図せず変化しません。
- 野手能力分布も before / after の全体サマリーで一致しています。

## 残っている許容差

- 架空球団用のafter差分は、先発: 球速 -1.132 / コントロール -1.380 / スタミナ +0.583、中継ぎ: 球速 -0.552 / コントロール +1.474 / スタミナ +1.309、抑え: 球速 -0.776 / コントロール +1.001 / スタミナ -0.875 です。
- いずれも指定された合格目安内で、球種数・総変化量の差分は調整前から変えていません。

## 完成扱い判断

架空球団用投手の役割別差分は十分縮小し、ドラフト候補用・助っ人外国人用のカテゴリ差も意図した範囲に残っています。したがって、現時点では**投手基本能力は完成扱い可能**です。
