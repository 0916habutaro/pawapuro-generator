# 通常特殊能力バランス検証レポート

## 調整内容の概要

- `ranked_specials` は通常特殊能力候補から除外したまま、通常特殊能力の `weight` と生成時の軽い出現補正のみを調整した。
- 汎用・個性系（国際大会、人気者、ムード、チームプレイ、調子極端、投球位置）はレア寄りに下げた。
- 強力な青特・青赤特はベース weight を控えめにし、能力値や選手タイプが合う場合だけ少し出やすくした。
- 赤特はベース出現をやや確保しつつ、低能力条件でだけ加点されるようにした。

## 検証条件

- 生成件数: 投手5,000件 / 野手5,000件（合計10,000件）
- 通常特殊能力 合計出現数: 16,476件
- 通常特殊能力 平均数: 1.648個/人
- ランク系特殊能力 平均数: 7.553個/人
- 検証コマンド: `python3 scripts/validate_special_balance.py` と同じ固定 seed・カテゴリローテーションで集計

## 追加した主な補正ルール

| 対象 | 補正概要 |
|---|---|
| 野手・長距離系 | `長距離砲` とパワー70/80以上で長打系を加点、パワー55未満で減点 |
| 野手・巧打系 | `巧打型` とミート70以上で巧打系を加点、ミート55未満でアベレージヒッター系を減点 |
| 野手・俊足系 | `俊足型` と走力70以上で走塁・盗塁・内野安打系を加点、走力55未満で減点 |
| 野手・守備系 | `守備職人`、守備力/捕球70以上で守備系を加点、低守備・低捕球で守備職人を減点 |
| 野手・強肩系 | `強肩型`、肩力70以上でレーザービーム・送球系を加点、肩力55未満で減点 |
| 捕手向き | 捕手で加点し、捕手以外では明確な捕手向き通常特殊能力を出さない |
| 投手・速球系 | `速球派` や球速150km/h以上で速球系を加点、145km/h未満で強い速球系を減点 |
| 投手・制球系 | `技巧派` やコントロール70以上で制球・緩急系を加点 |
| 投手・低制球赤特 | コントロール45未満で四球・荒れ球・乱調・ボール先行を加点、高コントロールで減点 |
| 投手・変化球系 | `変化球派`、球種数3以上、総変化量10以上で変化球系を加点 |
| 投手・スタミナ/役割 | 先発・スタミナ型で長いイニング向き特殊能力を加点、抑えでは一部を減点 |

## 確認観点まとめ

| 観点 | 結果 |
|---|---|
| 通常特殊能力平均1.648は許容範囲内か | 1.5〜2.2の範囲内 |
| 上位が汎用/個性系に偏っていないか | 上位30に国際大会○/×、人気者、ムード○/×は入っていない |
| タイプ補正が固定化していないか | 各タイプ上位に傾向は出ているが、1位でもタイプ対象人数比で概ね数%台で固定化はしていない |
| 赤特が低能力選手に自然に寄っているか | 四球・ボール先行・乱調・エラーは寄っている。三振と荒れ球は追加調整候補 |
| 通常特殊能力への ranked_specials 混入 | 0件 |
| 投手/野手専用特殊能力の混入 | 0件 |
| 同一 group の通常特殊能力重複 | 0件 |
